# express-locallibrary-tutorial

[Mozilla's tutorial app on Express/Node](https://developer.mozilla.org/en-US/docs/Learn/Server-side/Express_Nodejs)
The app demonstrates implementing CRUD features using Express, DB using MongoDB on [Atlas](https://www.mongodb.com/cloud/atlas) and dynamically generated views using [PUG](https://pugjs.org).
App is deployed in Heroku so please wait for about 10 seconds to initialize [demo website](http://locallibrary.pioclaudio.com/)
